update()

When editing, we don‘t actually allow users to change their own addresses. This is to prevent users changing addresses that are associated with old invoices.

Alternatively, invoices could be redesigned so that they contain the addresses as full strings instead of address objects, but that would introduce duplication on that level. This is a clear trade-off situation.
